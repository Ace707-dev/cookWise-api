# CookWise (WIP)

Una aplicación para buscar recetas de cocina tomando en cuenta ingredientes disponibles. Usa SwiftUI como interfaz para iOS/macOS y un backend en Vapor con base de datos MySQL.

---

## Funcionalidades

-  Buscar recetas por ingredientes
-  Ver detalles de cada receta (ingredientes, pasos)
-  Guardar recetas favoritas
-  CRUD de recetas e ingredientes
-  Backend API REST construida con Vapor + MySQL
-  Vista previa de componentes en tiempo real con SwiftUI

---

##  Tecnologías utilizadas

### Frontend
- [SwiftUI](https://developer.apple.com/xcode/swiftui/)  
- Arquitectura [MVVM](https://en.wikipedia.org/wiki/Model–view–viewmodel)

### Backend
- [Vapor](https://vapor.codes/) (Swift backend framework)
- [MySQL](https://www.mysql.com/) en la nube
